Student-Learning-Log:

This web project helps you to Make your own Learning Log, and keep a list of the topics you're learning about. 
Whenever you learn something new about a topic, make an entry summarizing what you've learned.

This project is developed using:
1. DJango (For backend)
2. Bootstrap4
3. HTML (For frontend+style)
3. Heroku(To deploy it in server)
4. git (for version control)


live app link:

https://student-learning-log.herokuapp.com/
